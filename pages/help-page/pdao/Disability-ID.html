Citizen's Charter

REQUIREMENTS FOR DISABILITY CARD

Where:

  • Persons with Disabilities Office (City Hall)

Who may avail:

  • Person With Disabilities

Requirements

  • Duly accomplished PDAO Application form
  • Duly accomplished DOH-PRPWD Form
  • Latest 1x1 and 2x2 ID pictures
  • Latest Barangay Clearance or any Government Issued ID with the same address
  • Authorization Letter if the applicant is not available
  • School ID or Birth Certificate if applicant is still a minor
  • For Apparent Disability - Certification from PDAO
  • For Non-Apparent Disability - Certification from Medical Specialists for special cases and any government and private doctors for other cases. Applicant for visual disability should bring certification issued by an Ophthalmologist
  • Duly accomplished PDAO Application form
  • Duly accomplished DOH - PRPWD form
  • Latest 1x1 and 2x2 ID pictures
  • Old/ Expired Disability ID
  • Duly accomplished PDAO Application form
  • Latest 2x2 ID pictures
  • Affidavit of Loss
  • Duly accomplished PDAO Application form
  • If newly married - Photocopy of Marriage Certificate
  • If annulled - Annotated Marriage Certificate issued by PSA
  • If widowed - Death Certificate of Spouse
  • Duly accomplished PDAO Application form
  • Original copy of Barangay Clearance
  • Old Disability ID
Notes:
  • MONDAY to FRIDAY, 8:00 AM - 5:00 PM ng (maliban kung piyesta opisyal o suspendido ang pasok sa opisina)

Steps

Isumite ang mga requirements sa: Persons with Disabilities Office (PDAO) - Ground Flr. R. Papa Bldg. Taguig City Hall Compound, Brgy. Tuktukan, Taguig City. • Tel no: 8642- 3590
